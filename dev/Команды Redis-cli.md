Положить данные в редис:
```redis
set test hello
```

Прочитать
```redis
get test
```

Удалить ключ
```redis
del test
```

Keys - команда, которая ищет ключи по маске. Используется итератор для поиска, поэтому рекомендуется не использовать, так как сразу оказывает влияние на производительность.
